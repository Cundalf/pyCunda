Archivos:
dorganizer.py
config.ini

Config:
->general
-->daysold -> Cuantos dias conservar. Configuracion necesaria cuando "ban/delold" este en 1.

->ban
-->image -> Filtrar imagenes
-->zip -> Filtrar comprimidos
-->audio -> Filtrar audio
-->video -> Filtrar videos
-->iso -> Filtrar iso
-->exec -> Filtrar ejecutables
-->prog -> Filtrar archivos asociados a programacion
-->docs -> Filtrar documentos
-->model -> Filtrar archivos asociados a modelos.
-->delold  -> Activar el borrado de archivos antiguos. Periodo configurable en "general/daysold".

->names
-> image -> Ejemplo: Images
-> zip -> Ejemplo: Zips
-> audio -> Ejemplo: Audios
-> video -> Ejemplo: Videos
-> iso -> Ejemplo: Isos
-> exec -> Ejemplo: Exes
-> prog -> Ejemplo: Prog
-> docs -> Ejemplo: Docs
-> model -> Ejemplo: Models
